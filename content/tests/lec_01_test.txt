1. Данные — это:
(x) представление информации в материальной форме;
( ) знания относительно фактов, событий, вещей, идей и понятий, которые в определённом контексте имеют конкретный смысл;
( ) сведения, воспринимаемые человеком как отражение фактов материального или духовного мира в процессе коммуникации.

2. Сколькими символами (цифрами) записываются троичные значения:
( ) 8;
( ) 4;
(х) 3.

3. В шестнадцатичечной системе арабские цифры дополнены следующими символами:
( ) A, B, C;
(x) A, B, C, D, E, F;
( ) A, B, C, D, E.

4. В унарной системе записи чисел используется:
( ) 10 цифр;
( ) 2 цифры;
(x) 1 цифра.

5. Бит может принимать:
(x) два значения: 0 и 1;
( ) значения в диапазоне от 0 до 255;
( ) значения в диапазоне от -127 до 128.

6. Байт состоит из:
( ) четырёх бит;
(x) восьми бит;
( ) шестнадцати бит.

7. Cтандарт ASCII касается:
( ) кодирования чисел;
(x) кодирования текста;
( ) использования носителей информации (жесткие диски, флешки и прочие).

8. В кодировке UTF-8 для представления символа:
(x) используется код переменной длины;
( ) используется фиксированное количество бит;
( ) не используется битовое представление.

9. Код переменной длины означает, что:
(x) для представления разных символов используется разное количество бит в зависимости от частоты их появления в тексте;
( ) размер закодированных данных в целях безопасности всегда разного объёма;
( ) размер данных может произвольно изменяться в зависимости от обстоятельств.
